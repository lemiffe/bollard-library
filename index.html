Country Bollards SVG Library

A free collection of GeoGuessr bollards in SVG format.

Prominent kilometre markers and signs are used for places with less predominant bollards. A range of standard sizes have been used to ensure they can be placed proportionately on maps and other media without distortion. 

npm install bollard-library

For more instructions check out our GitHub.
